** SI NO TE SALE LA VERSION Y TE SALE ERROR, EJECUTA EL SIGUIENTE COMANDO Y VUELVE A CONSULTAR LA VERSION HASTA QUE TE SALGA LA MISMA CONFIGURACION **
** npm install -g @angular/cli@16.2.10 **

Una vez que tengas las versiones, para levantar el sitio despues de pull es necesario tener la carpeta de node_modules en PortalGalleto

En caso de que no:

1.- En la terminal, aseguramos de estar dentro de la carpeta PortalGalleto
2.- Ejecutamos: <npm install -f>

Una vez terminado, dentro de PortalGalleto ejecutamos:

<ng serve>

Esto levantara el sitio y en el navegador entramos a http://localhost:4200/#/login/login
